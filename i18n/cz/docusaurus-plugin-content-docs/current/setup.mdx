# Příprava prostředí

## Příkazová řádka

Příkazová řádka je důležitým nástrojem při vývoji moderních webových aplikací.
Často je využívána pro tvorbu nových projektů, pro správu jejich modulů a také pro
správu jejich zdrojových kódů. Různé operační systémy nabízí jejich vlastní nástroje
(například systém Windows poskytuje [Command Prompt](https://en.wikipedia.org/wiki/Cmd.exe)
and [PowerShell](https://en.wikipedia.org/wiki/PowerShell), zatímco systémy na bázi Unixu
nabízí různé druhy [shells](https://en.wikipedia.org/wiki/Unix_shell)), ale v našem případě
budeme používat `bash`, který je _multiplatformní_.

- Na Unixových systémech by `bash` (nebo jedna z jeho variant jako například `zsh`) měl být
k dispozici automaticky, a pokud ne, můžete jej nainstalovat z https://www.gnu.org/software/bash.
- V systémech Windows bude `bash` nainstalován jako součást `git`, nástroje pro správu zdrojového
kódu, na který se podíváme v dalším odstavci.

## Source code management

Another essential part of the modern software development, especially for web, is the source code
management tool [git](https://git-scm.com). Go to the [Downloads](https://git-scm.com/downloads)
section, and install the latest release for your platform. As mentioned in the previous section,
the Windows installer comes with the `bash` CLI as well.

To check whether `git` is installed and available, try running the following command in `bash`:

```bash
git --version
```

## Editor

For the coding itself you're welcome to use whichever editor you like the most.
If you're open to suggestions, we recommend [Visual Studio Code](https://code.visualstudio.com)
which is free, cross-platform, and fast. Simply install the latest stable build
for your platform.

:::tip
We've also built an extension for Visual Studio Code that allows you to access
some of the Forge services and data directly from the editor:
[vscode-forge-tools](https://marketplace.visualstudio.com/items?itemName=petrbroz.vscode-forge-tools).
:::
